Change 4 — Add URL scan AJAX handler
File: includes/class-rairo-ajax.php
In register_ajax_handlers, add this line with the other add_action calls:
phpadd_action( 'wp_ajax_rairo_scan_url', array( $this, 'scan_url' ) );
Then add this new public method to the class (place it directly after the analyze_content method):
phppublic function scan_url() {
    $this->verify_nonce();
    $this->check_capability();

    $url      = isset( $_POST['url'] ) ? esc_url_raw( wp_unslash( $_POST['url'] ) ) : '';
    $industry = isset( $_POST['industry'] ) ? sanitize_text_field( wp_unslash( $_POST['industry'] ) ) : '';
    $post_id  = isset( $_POST['post_id'] ) ? absint( $_POST['post_id'] ) : 0;

    if ( empty( $url ) ) {
        wp_send_json_error( array( 'message' => __( 'A URL is required.', 'rain-os-ai-readability-optimizer' ) ) );
    }

    if ( ! filter_var( $url, FILTER_VALIDATE_URL ) ) {
        wp_send_json_error( array( 'message' => __( 'Please enter a valid URL including http:// or https://', 'rain-os-ai-readability-optimizer' ) ) );
    }

    $result = $this->api_client->scan_url( $url, $industry );

    if ( is_wp_error( $result ) ) {
        wp_send_json_error( array(
            'message' => $result->get_error_message(),
            'code'    => $result->get_error_code(),
        ) );
    }

    $parsed = $this->api_client->parse_analysis_response( $result );

    if ( $post_id > 0 ) {
        $this->save_analysis_to_history( $post_id, $parsed );
    }

    $usage_info = $this->api_client->get_last_usage_info();

    wp_send_json_success( array(
        'analysis'     => $parsed,
        'usage'        => $usage_info,
        'url_scanned'  => $url,
        'technical'    => isset( $parsed['technical_signals'] ) ? $parsed['technical_signals'] : null,
        'tech_recs'    => isset( $parsed['technical_recommendations'] ) ? $parsed['technical_recommendations'] : array(),
    ) );
}

Change 6 — Wire up the URL scan in JavaScript
File: assets/js/admin.js
Add the following JavaScript. If there is already an analyse button click handler, add the URL scan logic alongside it — do not replace it.
javascript// ── Scan mode toggle ──────────────────────────────────────────────────────────
document.addEventListener('DOMContentLoaded', function () {

    const modeBtns   = document.querySelectorAll('.rain-os-mode-btn');
    const urlArea    = document.getElementById('rain-os-url-input-area');
    const pasteArea  = document.getElementById('rain-os-content-input'); // adjust selector if needed

    modeBtns.forEach(function (btn) {
        btn.addEventListener('click', function () {
            modeBtns.forEach(function (b) { b.classList.remove('active'); });
            btn.classList.add('active');

            const mode = btn.dataset.mode;
            if (mode === 'url') {
                if (urlArea)   urlArea.style.display   = 'block';
                if (pasteArea) pasteArea.style.display = 'none';
            } else {
                if (urlArea)   urlArea.style.display   = 'none';
                if (pasteArea) pasteArea.style.display = 'block';
            }
        });
    });

    // ── URL scan submit ───────────────────────────────────────────────────────
    const analyseBtn = document.getElementById('rain-os-analyse-btn'); // adjust if needed
    if (analyseBtn) {
        analyseBtn.addEventListener('click', function (e) {
            const activeMode = document.querySelector('.rain-os-mode-btn.active');
            if (!activeMode || activeMode.dataset.mode !== 'url') return; // let existing paste handler run

            e.preventDefault();
            e.stopImmediatePropagation();

            const urlInput = document.getElementById('rain-os-scan-url');
            if (!urlInput || !urlInput.value.trim()) {
                alert(rairoData.i18n.url_required || 'Please enter a URL to scan.');
                return;
            }

            const url      = urlInput.value.trim();
            const industry = document.getElementById('rairo_industry') ? document.getElementById('rairo_industry').value : '';
            const postId   = rairoData.post_id || 0;

            // Show loading state
            analyseBtn.disabled = true;
            analyseBtn.textContent = rairoData.i18n.scanning || 'Scanning URL…';

            jQuery.ajax({
                url:    rairoData.ajax_url,
                type:   'POST',
                data: {
                    action:   'rairo_scan_url',
                    nonce:    rairoData.nonce,
                    url:      url,
                    industry: industry,
                    post_id:  postId,
                },
                success: function (response) {
                    analyseBtn.disabled = false;
                    analyseBtn.textContent = rairoData.i18n.analyse || 'Analyse';

                    if (response.success && response.data) {
                        // Dispatch same event the paste analysis uses so results display works unchanged
                        const event = new CustomEvent('rairo:analysisComplete', { detail: response.data });
                        document.dispatchEvent(event);

                        // Show technical signals if the results panel supports it
                        if (response.data.technical && window.rairoDisplayTechnicalSignals) {
                            window.rairoDisplayTechnicalSignals(response.data.technical, response.data.tech_recs);
                        }
                    } else {
                        const msg = (response.data && response.data.message) ? response.data.message : 'URL scan failed.';
                        alert(msg);
                    }
                },
                error: function () {
                    analyseBtn.disabled = false;
                    analyseBtn.textContent = rairoData.i18n.analyse || 'Analyse';
                    alert('Network error during URL scan. Please try again.');
                }
            });
        });
    }
});

Change 9 — Handle OAuth callback in the plugin
File: rain-os-ai-readability-optimizer.php (in the main plugin class or the loader)
Add a WordPress admin_init hook that checks if the user has just returned from OAuth (the URL will contain ?rairo_auth=1&apiKey=...) and saves the key automatically:
phpadd_action( 'admin_init', function () {
    if ( isset( $_GET['rairo_auth'] ) && '1' === $_GET['rairo_auth'] && isset( $_GET['apiKey'] ) ) {
        if ( current_user_can( 'manage_options' ) ) {
            $api_key = sanitize_text_field( wp_unslash( $_GET['apiKey'] ) );
            if ( ! empty( $api_key ) ) {
                update_option( 'rairo_api_key', $api_key );
                wp_redirect( admin_url( 'admin.php?page=ai-readability-settings&connected=1' ) );
                exit;
            }
        }
    }

    // Show success notice after OAuth redirect
    if ( isset( $_GET['connected'] ) && '1' === $_GET['connected'] ) {
        add_action( 'admin_notices', function () {
            echo '<div class="notice notice-success is-dismissible"><p>';
            esc_html_e( 'Rain OS connected successfully. You are ready to analyse content.', 'rain-os-ai-readability-optimizer' );
            echo '</p></div>';
        } );
    }
} );
