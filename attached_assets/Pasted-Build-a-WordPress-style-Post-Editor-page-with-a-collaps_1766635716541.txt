Build a WordPress-style Post Editor page with a collapsible AEO Analysis sidebar. Use React, TailwindCSS, Recharts, and react-circular-progressbar.

Design System:

Dark navy background (#0f172a), cards (#0c1222), cyan accents (#22d3ee)
Text: Primary (#f8fafc), Secondary (#94a3b8), Muted (#64748b)
Status: Success (#10b981), Warning (#f59e0b), Error (#ef4444)
LAYOUT STRUCTURE:
Split view with flexbox, full viewport height minus header:

Left: Content Editor (flex-1, min-width-0)
Toolbar: 56px height with "Edit Post" label, Draft badge, Heatmap toggle, Save/Publish buttons, sidebar collapse button
Editor area: contentEditable div with large title (text-4xl font-serif) and body content (prose-lg), max-width 768px, centered
Right: Analysis Sidebar (400px fixed width, conditionally rendered)
Slide-in animation from right when opened
Dark background with cyan accent header showing "Rain OS Analysis" + version badge
SIDEBAR TAB SYSTEM (4 tabs):
Use a tabbed interface with these sections:

Tab 1: Overview

Main Score Card: Circular progress indicator (react-circular-progressbar) showing overall score (0-100), cyan color, "GOOD/OK/NEEDS WORK" badge below
"Run Full Analysis" button with RefreshCw icon
3 Pillar Cards stacked vertically:
AI Readability (Microscope icon)
Digital Authority (ShieldCheck icon)
Conversion Readiness (Target icon)
Each shows icon, label, and score value
Critical Issues section: List of recommendation cards with left border color based on severity (red=critical, amber=warning)
Tab 2: Metrics

Horizontal Bar Chart (Recharts BarChart, layout="vertical", 200px height) showing 7-9 metrics:
Semantic Clarity, Logical Structure, Entity Recognition, Citation Readiness, AEO Alignment, QA-format Detection, Schema Extraction
Bar colors based on score: green (>75), amber (50-75), red (<50)
Authorship Provenance Card:
Gradient background (blue-900/30 to cyan-900/30)
Shows verification hash (monospace font), timestamp, "VERIFIED" badge
Engine version display
Tab 3: Actions

2x2 grid of action buttons:
Suggest Titles (Wand2 icon)
Meta Description (FileText icon)
Summarize (Fingerprint icon)
Rewrite Selection (RefreshCw icon)
Usage Quota section:
Progress bar showing queries used vs limit (e.g., "850 / 2,000")
"Reset in X days" text
Tab 4: History

List of previous analysis entries, each showing:
History icon in circle
Score value + "AEO Score" label
Timestamp
Delta change (+X points) in green if positive
ChevronRight arrow
Hover effect: cyan background tint
INTERACTIVE FEATURES:

Sidebar Toggle: Button to show/hide the 400px sidebar with slide animation
AI Heatmap Toggle: On/Off button that highlights specific phrases in the editor content with colored underlines (yellow for warnings, green for good sections)
Tab Navigation: Active tab shows cyan background/border, inactive tabs are muted gray
Hover States: All cards and buttons have hover:bg-cyan/10 and slight border glow effects
