AI-powered content analysis for Answer Engine Optimization (AEO) and Generative Engine Optimization (GEO).

== Description ==

Rain OS SEO Analyzer is a premium WordPress plugin that provides AI-powered content analysis to optimize your content for answer engines and generative AI systems.

= Key Features =

* **AI Readability Analysis** - Measure how well AI systems can understand and process your content
* **Digital Authority Scoring** - Assess your content's credibility signals that AI engines trust
* **Conversion Readiness** - Evaluate how effectively your content drives user action
* **Industry-Specific Analysis** - Get tailored recommendations based on your content's industry
* **Beautiful Dashboard** - Modern, premium UI matching the quality of top SEO plugins
* **Real-time Results** - Instant analysis with animated score displays
* **Usage Tracking** - Monitor your analysis usage with visual progress indicators

= The Five Pillars of AI Readability =

1. **AI Readability** - How well AI systems can understand and process your content structure
2. **Digital Authority** - Your content's credibility signals that AI engines trust and prioritize
3. **Conversion Readiness** - How effectively your content drives user action and engagement
4. **Product Discoverability** - How easily your product or service can be found through AI search
5. **RAG Readiness** - How well content is optimized for AI retrieval and synthesis systems

= Requirements =

* WordPress 5.8 or higher
* PHP 7.4 or higher
* Rain OS API account and API key

== Installation ==

1. Upload the `rain-os-seo` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Rain OS > Settings to configure your API endpoint and API key
4. Start analyzing your content!

= Manual Installation =

1. Download the plugin zip file
2. Log in to your WordPress admin panel
3. Navigate to Plugins > Add New > Upload Plugin
4. Choose the zip file and click Install Now
5. Activate the plugin
6. Configure your API settings under Rain OS > Settings

== Frequently Asked Questions ==

= Where do I get an API key? =

You can obtain an API key by signing up for a Rain OS account at https://rain-os.com

= What post types are supported? =

By default, the plugin works with Posts and Pages. You can enable additional post types in the General Settings tab.

= How many analyses can I perform? =

The number of analyses depends on your Rain OS subscription plan. Your usage is displayed in the dashboard and meta box.

= Is my content secure? =

Yes, your content is transmitted securely via HTTPS to our servers for analysis. We do not store your content after analysis is complete.
