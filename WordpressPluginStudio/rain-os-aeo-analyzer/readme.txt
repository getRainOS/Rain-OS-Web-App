AI-powered Answer Engine Optimization (AEO) and Generative Engine Optimization (GEO) for WordPress.

== Description ==

Rain OS AEO Analyzer helps you optimize your content for AI-powered answer engines like Google AI Overviews, Perplexity, ChatGPT, Claude, and Gemini.

= Three Pillars of Analysis =

**AI Readability** - Measures how easily AI systems can parse and understand your content:
* Semantic Clarity: Clear, unambiguous language
* Readability Score: Processing ease for both humans and AI
* Logical Structure: Proper heading hierarchy and organization

**Digital Authority** - Assesses the credibility and trustworthiness of your content:
* Entity Recognition: Links to known knowledge graph entities
* Citation Readiness: Content that can be quoted and attributed
* Schema Extraction: Structured data opportunities

**Conversion Readiness** - Evaluates how well your content drives engagement:
* AEO Alignment: Direct answer optimization
* QA-Format: Question and answer structure
* Metadata Audit: Schema and HTML verification

= Key Features =

* **Content Analyzer**: Full-featured editor with formatting toolbar and real-time analysis
* **AI Heatmap**: Color-coded highlighting showing pillar relevance
* **Dashboard**: KPI cards with speedometer gauges and performance charts
* **Score History**: Track your content optimization progress over time
* **Pillar Breakdown**: Detailed subcategory analysis with horizontal bars
* **Content Signals**: Word count vs. score distribution scatter plots
* **Quick Tools (Pro)**: AI-powered title suggestions, meta descriptions, summarization, and rewriting
* **Local Content Audit**: WordPress-powered checks without API calls
* **Notification System**: Stay updated on analysis completion and usage

= Requirements =

* WordPress 5.8 or higher
* PHP 7.4 or higher
* Rain OS API key (get one at app.getrainos.com)

== Installation ==

1. Upload the `rain-os-aeo-analyzer` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Rain OS > Settings and enter your API key
4. Start analyzing your content!

== Frequently Asked Questions ==

= What is AEO? =

AEO (Answer Engine Optimization) is the practice of optimizing your content to appear in AI-powered answer engines and featured snippets. It focuses on making your content easily understandable and citable by AI systems.

= Do I need an API key? =

Yes, Rain OS AEO Analyzer requires an API key to perform AI-powered analysis. You can get a free API key at app.getrainos.com.

= Is there a free plan? =

Yes! We offer a free tier with limited monthly analyses. Upgrade to Business or Pro for more analyses and advanced features.

= What content can I analyze? =

You can analyze any text content including blog posts, pages, product descriptions, and more. The analyzer works with both the Content Analyzer editor and your existing WordPress content.

= How are scores calculated? =

Scores are calculated using AI analysis across three pillars: AI Readability, Digital Authority, and Conversion Readiness. Each pillar has subcategories that contribute to the overall score.
